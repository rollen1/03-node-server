# Nodejs Server

_40 group project_

## Project features

-   Node.js: http
-   Static file serving

## Running the server

In `dev` mode:

```
node ./index.js
NODE_ENV=dev node ./index.js
NODE_ENV=test node ./index.js
NODE_ENV=pomidoras node ./index.js
```

In a specific (pre-defined) `prod` mode:

```
NODE_ENV=prod node ./index.js
```
